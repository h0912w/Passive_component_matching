# 저항 표기 가이드 (Resistor Notation Guide)

> 역할: 사람이 읽는 저항 표기 규칙 레퍼런스. 파서 코드 구현 시 참고.

---

## 1. IEC 60062 RKM 표기법

저항값을 소수점 없이 표기하기 위해 단위 문자를 소수점 위치에 삽입한다.

| 표기 | 의미 | 계산 |
|---|---|---|
| `4R7` | 4.7 Ω | 4 + 0.7 |
| `1R0` | 1.0 Ω | 1.0 |
| `0R` | 0 Ω (점퍼) | 0 |
| `1K` | 1,000 Ω | 1 × 1000 |
| `4K7` | 4,700 Ω | 4.7 × 1000 |
| `1M` | 1,000,000 Ω | 1 × 1,000,000 |
| `2M2` | 2,200,000 Ω | 2.2 × 1,000,000 |

**규칙**: `[숫자][단위문자][숫자]` 형식에서 단위 문자 앞뒤 숫자를 합쳐 소수점으로 해석.

## 2. 패키지 표기

### 2.1 메트릭 vs 인치

```
메트릭 표기: 실제 물리 크기(mm) 기반
  1608 → 1.6mm × 0.8mm

인치 표기: 인치 단위 근사값 기반
  0603 → 0.06" × 0.03" ≈ 1.6mm × 0.8mm
```

**동일한 부품이 두 가지 표기를 가짐:**
```
1608(메트릭) = 0603(인치) = 1.6 × 0.8 mm
2012(메트릭) = 0805(인치) = 2.0 × 1.2 mm
3216(메트릭) = 1206(인치) = 3.2 × 1.6 mm
```

### 2.2 현장 관행
- **"0603"** 단독: 인치 기준으로 해석 (1.6×0.8mm = 메트릭 1608)
- **"1608"** 단독: 메트릭 기준으로 해석 (1.6×0.8mm = 인치 0603)
- Mouser 검색 시 양쪽 표기를 OR 조건으로 검색

## 3. 오차 표기

### 3.1 % 표기 (가장 명확)
```
1%, ±1%, +/-1%  → 모두 동일
5%, ±5%
0.1%, 0.5%
```

### 3.2 IEC 문자코드
```
F → ±1%    (가장 흔함)
J → ±5%    (가장 흔함)
B → ±0.1%
C → ±0.25%
D → ±0.5%
G → ±2%
K → ±10%
M → ±20%
```

> 주의: `K`와 `M`은 저항값 단위와 중의성이 있음

## 4. 전력 표기

### 4.1 분수 표기 (현장 가장 흔함)
```
1/8W = 0.125W
1/4W = 0.25W
1/2W = 0.5W
```

### 4.2 소수점 표기
```
0.125W = 1/8W
0.25W = 1/4W
```

### 4.3 밀리와트
```
125mW = 0.125W = 1/8W
250mW = 0.25W = 1/4W
```

## 5. 표기 변형 대응 예시

| 입력 | 정규화 결과 |
|---|---|
| `4R7 1608 5% 0.25W` | 4.7Ω, 1608M/0603I, ±5%, 0.25W |
| `4.7k/1608/J/0.25W` | 4700Ω, 1608M/0603I, ±5%, 0.25W |
| `4k7_0603_F_125mW` | 4700Ω, 1608M/0603I, ±1%, 0.125W |
| `4.7kohm 0603 ±5% 1/4W` | 4700Ω, 1608M/0603I, ±5%, 0.25W |
