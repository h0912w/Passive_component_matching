## 품질 분석

### 정상 데이터

- **저항값**: E24/E96 계열 표준값 대부분 포함. 실용 범위(1Ω ~ 10MΩ) 양호.
- **패키지**: 5대 주요 패키지(0402/0603/0805/1206/2512) 포함.
- **오차**: 실용 범위 전체 포함 (0.05% ~ 5%).

### 이상값 주의

**전력 이상값**: 아래 값들은 저항이 아닌 다른 부품이 섞인 결과로 추정됨.

| 이상값 | 추정 원인 |
|---|---|
| 15W, 16W, 63W, 125W | 고전력 세멘트/와이어와운드 저항 또는 파싱 오류 |

→ 역검증(`validator.ts`)에서 패키지 + 전력 조합으로 자동 필터링됨. 별도 조치 불필요.

### 파싱 커버리지 (추정)

- Description에서 저항값 추출 성공률: ~85% (IEC RKM + ohms + K/M/R 패턴)
- 파싱 실패 부품(저항값 null)은 DB에서 제외됨

## 수집 방법 요약

```
방식: Cloudflare Worker 임시 /collect-db 엔드포인트
검색 키워드:
  - chip resistor 0402 (≤150건)
  - chip resistor 0603 (≤150건)
  - chip resistor 0805 (≤150건)
  - chip resistor 1206 (≤150건)
  - chip resistor 2512 (≤150건)
스펙 추출: Description + ManufacturerPartNumber regex 파싱
```

상세 수집 절차 → `/docs/mouser_db_guide.md`

## 재수집 기준

다음 상황에서 재수집을 검토한다:
- 파서 실패율이 높아 DB 교차 확인 효과가 낮은 경우
- Mouser 제품 라인업이 크게 변경된 경우 (반기 1회 검토 권장)
- 현재 DB에 없는 저항값 범위가 자주 검색되는 경우

재수집 시 이 파일의 **수집 이력** 섹션에 반드시 추가한다.
