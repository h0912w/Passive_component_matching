## 주의사항

### 동일 코드 다른 크기
아래 코드는 Metric과 Imperial에서 **다른 크기**를 의미:

| 코드 | Metric 의미 | Imperial 의미 |
|------|------------|--------------|
| 0402 | 0.4×0.2mm (극소) | 1.0×0.5mm (일반 소형) |
| 0603 | 0.6×0.3mm (극소) | 1.6×0.8mm (일반) |
| 2512 | 2.5×1.25mm (중형) | 6.3×3.2mm (대형) |

→ 파싱 시 기본값을 **metric**으로 간주 (한국/아시아 회로도에서 주로 사용)

## 가장 많이 사용되는 패키지 (빈도순)
1. **0402 (imperial)** = 1005 (metric) — 가장 범용, 소형 기기
2. **0603 (imperial)** = 1608 (metric) — 핸드솔더링 가능한 가장 작은 크기
3. **0805 (imperial)** = 2012 (metric) — 프로토타이핑, 교육용
4. **0201 (imperial)** = 0603 (metric) — 스마트폰 등 초소형
5. **1206 (imperial)** = 3216 (metric) — 고전력, 자동차용
