# Skill: reverse-validator

> 역할: 최종 PN 재조회 및 항목별 일치 판정 절차. 엣지케이스 발견 시 갱신.

## 역검증 절차

1. 선정된 PN으로 Mouser `/search/partnumber` 재조회
2. 재조회 실패 시 검색 결과 캐시 사용
3. 다음 항목 비교:
   - 저항값: ±1% 허용 (필수 항목)
   - 패키지: 메트릭/인치 양방향 일치 (선택)
   - 오차: ±0.5% 허용 (선택)
   - 전력: ±5% 허용 (선택)
4. 불일치 → 다음 후보 재시도 (최대 3회)
5. 모두 실패 → `VERIFICATION_FAILED`
